# RAK Rules: R01-ui-fundamentals

## Scope

Berlaku untuk rak ini dan semua buku di dalamnya.

## Required Structure

- README.md
- docs/RAK_RULES.md
- Folder buku dengan format Bxx-book-slug

## Allowed Changes

- Menambah buku baru dalam rak ini.
- Mengubah baseline penulisan untuk semua buku dalam rak ini.

## Forbidden Changes

- Mengubah aturan ROOT.
- Mengubah isi rak lain.

## Override

- Buku boleh memiliki aturan lebih spesifik di docs/BOOK_RULES.md.
